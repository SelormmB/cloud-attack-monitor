# Cloud Attack Monitor

A cloud-native cybersecurity monitoring dashboard designed to visualize suspicious traffic and attack activity in real time.

This project demonstrates modern cloud architecture using serverless technologies and infrastructure designed for scalability, security, and monitoring.

---

## Project Overview

Cloud Attack Monitor is a lightweight dashboard that displays simulated cyber attack data including attacker IP addresses, geographic origins, and daily attack counts.

The goal of this project is to demonstrate how modern cloud services can be used to build scalable monitoring systems similar to those used by security operations centers (SOC).

---

## Features

- Cyber attack monitoring dashboard
- Real-time attack statistics display
- IP address tracking
- Country-of-origin identification
- Serverless backend architecture
- Cloud-based data storage
- API-driven data retrieval
- Scalable infrastructure design

---

## Architecture

Frontend
- Static dashboard built with HTML/CSS
- Hosted via cloud object storage

Backend
- Serverless functions process incoming data
- API gateway exposes endpoints
- NoSQL database stores attack logs

Monitoring
- Logging and metrics through cloud monitoring tools
